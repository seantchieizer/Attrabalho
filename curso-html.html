.HTML

Entenda o que é HTML

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas web. Ele define títulos, parágrafos, links, imagens e muito mais.

Estrutura básica de uma página HTML

<!DOCTYPE html>
<html>
<head>
<title>Título da Página</title>
</head>
<body>
<h1> Olá, mundo! </h1>
<p> Esse é meu primeiro site. </p>
</body>
</html>

Principais tags HTML

<h1> a <h6>: Títulos
<p>: Parágrafos
<a href="">: Links
<img src="">: Imagens
<ul>, <ol>, <li>: Listas
<div> e <span>: Contêineres
